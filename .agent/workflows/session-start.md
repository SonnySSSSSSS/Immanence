# Session Start Workflow

// turbo-all

## Before Starting Any Work

1. Read the latest session summary document:
   ```
   cat "d:\Unity Apps\immanence-os\.agent\session-summary.md"
   ```

2. Read the immanence-rules workflow:
   ```
   cat "d:\Unity Apps\immanence-os\.agent\workflows\immanence-rules.md"
   ```

3. Confirm with user what to work on from the "Next Session" section.

## After Completing Work

1. Update `session-summary.md` with:
   - Date completed
   - Summary of what was done
   - Version number reached
   - Next session priorities

2. Commit the session summary to git (optional).
