# TASK-2026-03-27-A: Bundle Split Phase 2 — Lazy-Load HomeHub

**Created:** 2026-03-27
**Recommended Executor:** Claude Code / Codex
**Scope:** Convert `HomeHub` from an eager import to a `React.lazy` dynamic import in `App.jsx`, following the established pattern used for `WisdomSection`, `ApplicationSection`, and `PracticeSection`. Expected reduction: ~100–150 kB gzipped from the initial chunk.
**Flags:** `IMPLEMENT UNCOMMITTED, VERIFY FIRST` `NO COMMIT WITHOUT HUMAN APPROVAL`

---

## Goal

The initial `index-*.js` chunk still contains `HomeHub` (1,702 lines), which loads eagerly regardless of which section the user lands on. Deferring it behind a `React.lazy` boundary removes it from the initial chunk and reduces time-to-interactive.

This is Phase 2 of the strategy documented in `docs/BUNDLE_SPLIT_PLAN.md`. Phase 1 (PracticeSection) was completed in TASK-2026-03-26-C.

---

## Problem Statement

`src/App.jsx` contains an eager import of `HomeHub`:
```js
import { HomeHub } from "./components/HomeHub.jsx";
```

`HomeHub` is 1,702 lines and pulls in avatar rendering, dashboard panels, and curriculum access UI. It is included in the initial chunk even for users who navigate directly to a non-home section.

The established lazy-load pattern (already used for `WisdomSection`, `ApplicationSection`, `PracticeSection`) is:
```js
const WisdomSection = lazy(() =>
  import("./components/WisdomSection.jsx").then(m => ({ default: m.WisdomSection }))
);
```

`HomeHub` uses a named export, so the same `.then(m => ({ default: m.HomeHub }))` wrapper is required.

---

## Single Bounded Change

Convert `HomeHub` to a `React.lazy` import and wrap its render site in a `<Suspense>` boundary. No changes to `HomeHub.jsx` itself.

---

## Reuse-First Plan (Mandatory)

1. `src/App.jsx`
   - Remove the eager import line for `HomeHub`.
   - Add a lazy declaration immediately after the existing lazy declarations:
     ```js
     const HomeHub = lazy(() =>
       import("./components/HomeHub.jsx").then(m => ({ default: m.HomeHub }))
     );
     ```
   - Locate the existing render of `<HomeHub ...>`. Wrap it with `<Suspense fallback={<div />}>` using the same minimal fallback pattern already in place for other lazy sections.
   - `lazy` and `Suspense` are already imported from React — no new imports needed.

2. `src/components/HomeHub.jsx`
   - **No changes.**
