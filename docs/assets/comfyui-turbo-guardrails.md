# ComfyUI Turbo Guardrails – UI Assets

## Model class
- Turbo / lightning / fast checkpoints ONLY

## Hard limits
- Steps: 6–8 (NEVER > 8)
- CFG: 1.0 – 2.0 ONLY
- Sampler: default for model (do not override)
- No highres fix
- No post upscaling
- No text encoders beyond base
- **HARD RULE:** Save all generated images under `D:\Unity Apps\immanence-os\public\` only. Do not place image assets under `src/`.

## Failure indicators (REJECT output if any present)
- Mushy glow blobs
- Over-blended gradients
- Loss of anatomical clarity
- Text artifacts
- Over-sharpened halos

- Clear focal chakra(s)
- Recognizable human energy form
- Clean silhouette edges
- Visual hierarchy matches asset spec
