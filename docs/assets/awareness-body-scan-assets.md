# Awareness – Body Scan Asset Specification

## Global Style (ALL assets)
- Aspect ratio: 1:1
- Resolution: 1024x1024 (authoring), downscale to 768x768 for app
- Background: deep cosmic nebula, low contrast, dark edges
- Figure: gender-neutral energetic body, semi-transparent, ethereal
- No text, no UI glyphs, no labels
- Soft vignette, center-weighted composition
- Color language must match chakra system
- **HARD RULE:** All generated images must be stored in `D:\Unity Apps\immanence-os\public\` only. Never place image assets under `src/`.

## Chakra Visual Language
- Crown: violet / white glow (head)
- Third Eye: indigo glow (forehead)
- Throat: cyan glow (throat)
- Heart: green-gold glow (chest)
- Solar Plexus: golden-yellow glow (upper abdomen)
- Sacral: warm orange glow (lower abdomen)
- Root: deep red glow (pelvic base)
