# Immanence OS — Documentation Index

## Canonical Workspace

- Main folder: `D:\Unity Apps\immanence-os`
- Do not run or back up from `.claude-worktrees/...` — copy changes back to the main folder before running.
- Dev server: `npm run dev` → `http://localhost:5173/`
